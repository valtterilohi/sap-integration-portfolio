# REST to REST Integration

## Objective

Create an SAP Integration Suite iFlow that receives JSON data from a source API and forwards it to a target API.

## Architecture

```text
Source API
    |
    v
SAP CPI
    |
    v
Target API
```

## Technologies

- SAP Integration Suite
- Cloud Integration (CPI)
- REST
- JSON
- Message Mapping

## Skills Demonstrated

- API Integration
- Payload Transformation
- Monitoring
- Error Handling
